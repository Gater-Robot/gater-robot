What happens on-chain
Raw storage vs computed view — the innovation
rawBalanceOf()
Storage: stays at 30e18
// Only changes on transfer, mint, or settleDecay()
pendingDecay()
elapsed = now - lastUpdate
decay = (elapsed / 86400) * 1e18
// Computed, never stored
balanceOf() — what users see
balanceOf = raw - decay
= 30e18 - (days * 1e18)
// Pure view function = zero gas
✨ No gas cost for decay. It's a view function that computes on read.
